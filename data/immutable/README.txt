
Папка `immutable` содержит следующие файлы, связанные с параметрами и данными для различных расчетов:

1. **geodesia_const.yaml**
   - Формат: YAML
   - Описание: Этот файл содержит константы, используемые в геодезических расчетах. Включает такие параметры, как:
     - `0_OmegaZ`: 7.292115e-05
     - `1_fM`: 398600441800000.0

2. **oze_param.yaml**
   - Формат: YAML
   - Описание: Файл с параметрами для моделирования общеземного эллипсоида. Содержит следующие параметры:
     - `0_a_oze`: 6378136.0
     - `1_polar_compression`: 0.0033528037351842955
     - `2_R_sr`: 6371110.0
     - `3_e2_oze`: 0.0066943662

3. **stand_atmo_data.csv**
   - Формат: CSV
   - Описание: Этот файл содержит стандартные атмосферные данные в формате таблицы. Включает информацию о температуре и плотности воздуха на различных высотах.

4. **TCxM.csv**
   - Формат: CSV
   - Описание: Этот файл содержит данные полученные в результате обдувки реального прототипа. Включает информацию коэффициентах лобового сопротивления.
